0. What is Sabertooth for?

Sabertooth is for organizing files and sub-folders (you can set whether you want to organize files only/folders only later) in folder(s) you choose based on filters such as, files with .jpg extension/format will be placed in a folder named 'Picture'. You may also choose to exclude some file types by whitelisting in the 'Setings'.


It also has a Auto-Organize feature, which organizes your folder(s) every 30 minutes, you can set auto-organizing interval in the 'Settings'.

1. Getting to know the interface

Sabertooth's interface has been made to be simple and to look clean. I hope you won't have any difficulty learning the interface.

2. Setting up Sabertooth and changing it's settings

Some buttons may appear grayscale if there is an error, such as you trying to add an existing filter etc...

Adding a folder:

  1. Click the green plus button.
  2. Select the folder you want to organize from the dialog, and click OK.

Changing folder settings/preferences:

  1. Click the blue settings button.
  2. Select the 'Organized Folders' tab.
  3. Select the folder which you want to change it's settings.
  4. Select 'Organize All' if you want to organize all the files and the sub folders in that folder, select 'Only Files' to organize only the files in that folder leaving all the sub-folders untouched, select 'Only Folders' to organize only the folders in that folder leaving all the files untouched.
  5. Turn ON the 'Auto-Organize' to auto-organize this folder when 'Auto-Organizer' is ON, or OFF to exclude this folder

Removing a folder:

  1. Click the blue settings button.
  2. Select the 'Organized Folders' tab.
  3. Select the folder which you want to remove.
  4. Click the red 'Exclude Folder' button.

Organizing folder(s):

  1. Click the big folder button.
  2. Wait a second.

Changing the 'Auto-Organizer Interval':

  1. Click the blue settings button.
  2. Select the 'General' tab.
  3. Check the 'Turn on Auto-Organizer' box.
  4. Set the interval using the box next to the box you've just checked
  5. You may now uncheck the box, or leave it just as it is to keep 'Auto-Organizer' ON.

Make sure the 'Auto-Organizer' is off to prevent the folder you've just unorganized from being organized again.

Unorganizing a folder:

  1. Click the blue settings button.
  2. Select the 'Organized Folders' tab.
  3. Select the folder which you want to unorganize.
  4. Click 'Unorganize Folder' button.
  5. Click 'Yes' if you are sure.

Make sure the 'Auto-Organizer' is off to prevent the folders you've just unorganized from being organized again.

Unorganizing all folders:

  1. Click the blue settings button.
  2. Select the 'About & Help' tab.
  3. Click 'Unorganize All Folders' button.
  4. Click 'Yes' if you are sure.

Adding a filter:

  1. Click the blue settings button.
  2. Select the 'Filters' tab.
  3. Type the extension you want to organize in the box on the right. (Like this => .jpg)
  4. Select the sub folder you want this file type to group in.
  5. Click 'Add Filter' button.

Removing a filter:

  1. Click the blue settings button.
  2. Select the 'Filters' tab.
  3. Select the filter which you want to remove.
  4. Click the red 'Remove Filter' button.

Whitelisting a file type:

You must remove all the filters associated with this the file type you want to whitelist first

  1. Click the blue settings button.
  2. Select the 'Whitelist' tab.
  3. Type the extension you want to whitelist in the box on the right. (Like this => .jpg)
  4. Click 'Add to Whitelist' button.

Removing a whitelisted file type:

  1. Click the blue settings button.
  2. Select the 'Whitelist' tab.
  3. Select the whitelisted file type which you want to remove.
  4. Click the red 'Remove from Whitelist' button.

Adding a sub folder:

  1. Click the blue settings button.
  2. Select the 'Sub Folders' tab.
  3. Type the sub folder name you want to add in the box on the right.
  4. Click 'Add Sub Folder' button.

Removing a sub folder:

You must remove all the filters associated with this sub folder first

  1. Click the blue settings button.
  2. Select the 'Sub Folders' tab.
  3. Select the sub folder which you want to remove.
  4. Click the red 'Remove Sub Folder' button.

Adding a custom sub folder:

  1. Click the blue settings button.
  2. Select the 'Sub Folders' tab.
  3. Click the magnifying glass icon to browse for a folder, then click OK.
  4. Click 'Add Custom Sub Folder' button.

Removing a custom sub folder:

You must remove all the filters associated with this custom sub folder first

  1. Click the blue settings button.
  2. Select the 'Sub Folders' tab.
  3. Select the custom sub folder which you want to remove.
  4. Click the red 'Remove Custom Sub Folder' button.

Updating Sabertooth:

  1. Click the blue settings button.
  2. Select the 'Updates' tab.
  3. Check if updates are available by clicking 'Check for Updates' button.
  4. If updates are available, the button below will activate.
  5. To download the new version, click the button below.
  6. Now, Sabertooth will uninstall itself, while downloading the installer of the new version (you won't lose your settings)
  7. Launch the downloaded setup file and follow it's steps.

Resetting Sabertooth:

  1. Click the blue settings button.
  2. Select the 'About & Help' tab.
  3. Click 'Reset All Settings' button.
  4. Click 'Yes' if you are sure.
